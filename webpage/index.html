DeepCool Digital on Linux

A GTK app for DeepCool Digital USB/HID devices, built to monitor hardware, update the cooler display, and keep your setup running quietly in the background.

Built for the desktop

The app keeps the original DeepCool Digital workflow familiar while adding Linux packaging, local installation, tray presence, and background persistence.

01

Live data

Track CPU, GPU, pump, fan, temperature, usage, power, and frequency data when supported by your system.

02

Display modes

Choose what appears on the cooler panel and rotate between CPU usage and temperature with a custom interval.

03

Background mode

Install a systemd service and keep the device updated without leaving the main window open.

04

Linux packages

Use the portable AppImage or install the Debian package on Ubuntu, Debian, and compatible distributions.

Clean control

White interface, teal accents, device pages, settings, and live previews inspired by the official DeepCool software style.

Download for Linux

Ready-to-use packages are available for the first release. Pick AppImage for a portable run, or use the Debian package for a cleaner install on Debian, Ubuntu, and derivatives.

The app asks for administrator permission when needed because supported DeepCool devices are accessed through USB/HID. This project is community-made and is not affiliated with DeepCool.
